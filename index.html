ทุกคำถามที่คนรอเงินคืนอยากรู้

แตะที่หัวข้อเพื่อดูรายละเอียด

เมื่อคุณยื่น ภ.ง.ด.90/91 ระบบสรรพากรจะ เช็คข้อมูลที่คุณกรอกกับฐานข้อมูล ที่นายจ้าง บริษัทประกัน และธนาคารส่งมาแล้ว — ถ้าตรงกัน → อนุมัติเลย ถ้าไม่ตรง → ส่งให้เจ้าหน้าที่ตรวจ
⚡ ได้ใน 3 วัน
  • ตัวเลขรายได้ตรงกับระบบ
  • สิทธิลดหย่อนมีข้อมูลในระบบแล้ว
  • ไม่มีรายการผิดปกติ
⏳ รอนานกว่า
  • ตัวเลขไม่ตรงกับระบบ
  • บริษัทประกัน/บลจ. ยังไม่ส่งข้อมูล
  • ไม่ได้แนบเอกสาร
  • มีรายการต้องตรวจก่อน
1
ระบบตรวจพบข้อมูลไม่ตรง → flag รายการไว้
2
เจ้าหน้าที่ต้องตรวจด้วยตัวเอง — ผู้ยื่นมีล้านราย เจ้าหน้าที่มีน้อย จึงใช้เวลา
3
อาจส่งหนังสือขอเอกสารเพิ่ม — ถ้าไม่ตอบ ยิ่งล่าช้า
4
ตรวจสอบเสร็จ → อนุมัติคืนภาษี
อยู่ระหว่างรวบรวมข้อมูล

มีผู้เสียภาษีบางส่วนที่ข้อมูลตรงกับระบบทุกอย่าง แต่ยังถูก flag ให้ตรวจสอบ
อยู่ระหว่างสอบถามข้อมูลจากกองเทคโนโลยีสรรพากร

จะอัปเดตเร็วๆ นี้ — ปรึกษาเราได้เลย ถ้าอยากรู้ว่าเคสของคุณเป็นแบบไหน

🛡️ ประกัน
ประกันชีวิต / ประกันสุขภาพ⚠️ แจ้งบริษัทก่อน
  • กรมธรรม์ปี 2563 ขึ้นไป: แจ้งบริษัทประกันให้ส่งข้อมูลให้สรรพากร
  • กรมธรรม์ก่อนปี 2563: แนบใบเสร็จรับเงินหรือหนังสือรับรอง
ถ้าบริษัทไม่ส่ง สรรพากรจะไม่ให้สิทธิ์ แม้จ่ายจริง
📈 กองทุน
RMF / SSF / Thai ESG⚠️ แจ้ง บลจ. ก่อน
  • แจ้งบริษัทหลักทรัพย์จัดการกองทุน (บลจ.) ให้ส่งข้อมูลให้สรรพากร
ถ้า บลจ. ยังไม่ส่ง สิทธิ์จะถูกปฏิเสธ
กองทุนสำรองเลี้ยงชีพ (PVD)📎 แนบเอกสาร
  • ใบรับรองหรือใบแจ้งยอดเงินสมาชิก PVD จากนายจ้าง
👨‍👩‍👧‍👦 ครอบครัว
บิดา-มารดา📎 แนบเอกสาร
  • หนังสือรับรอง แบบ ลย.03 (ดาวน์โหลดจากเว็บสรรพากร)
บุตร📎 แนบเอกสาร
  • สูติบัตรบุตร
  • ผู้ชาย: ต้องแนบทะเบียนสมรสด้วย
คนพิการ / ทุพพลภาพ📎 แนบเอกสาร
  • แบบ ล.ย.04 + สำเนาบัตรประจำตัวคนพิการ
🏠 บ้าน
ดอกเบี้ยเงินกู้ซื้อบ้าน⚠️ แจ้งธนาคารก่อน
  • สัญญาปี 2564 ขึ้นไป: แจ้งธนาคารให้ส่งข้อมูลให้สรรพากร
  • สัญญาก่อนปี 2564: แนบหนังสือรับรองดอกเบี้ยจากธนาคาร
⚠️ ยื่นผิดประเภท = ต้องจ่ายภาษีเพิ่ม + เบี้ยปรับ ถ้ายื่นประเภทที่หักค่าใช้จ่ายได้สูงกว่าโดยไม่มีหลักฐาน สรรพากรจะปรับกลับและเรียกเก็บส่วนต่าง
40(1)เงินเดือน / บำนาญ
50% ไม่เกิน 100,000 บ.
นายจ้างรายงานเข้าระบบสรรพากรอัตโนมัติ
40(2)ค่านายหน้า / ค่าตอบแทน
50% ไม่เกิน 100,000 บ. (รวมกับ 40(1))
ค่านายหน้าประกัน · เบี้ยประชุม · ค่าตอบแทนสัญญาทั่วไป
40(6)วิชาชีพอิสระ
30–60% ตามสาขาวิชาชีพ
แพทย์ 60% · กฎหมาย/วิศวกร/บัญชี 30%
40(7)รับเหมา (จัดหาวัสดุเอง)
60% ไม่มีเพดาน
รับเหมาก่อสร้าง · ติดตั้งระบบ — ต้องจัดหาวัสดุในส่วนสำคัญ
⚠️ พิสูจน์ไม่ได้ → ปรับเป็น 40(2) + เบี้ยปรับ
40(8)ธุรกิจ / Freelance / อื่นๆ
40–85% ตามประเภท หรือหักตามจริง
ธุรกิจทั่วไป · แกร็บแท็กซี่ · ขายสินค้าออนไลน์
